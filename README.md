# E-commerce Analytics, ML & Recommendation System

## Overview
End-to-end data science project involving EDA, customer segmentation using K-Means, and a recommendation system.

## Dataset
Online Retail Dataset (UCI / Kaggle)

## Techniques Used
- Exploratory Data Analysis (EDA)
- Feature Engineering
- K-Means Clustering
- Collaborative Filtering (Cosine Similarity)

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Key Outcomes
- Identified high-value customer segments
- Discovered seasonal sales trends
- Built a product recommendation system
